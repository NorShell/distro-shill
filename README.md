- **Backend:** [Hono](https://hono.dev/)

- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Database:** [D1](https://developers.cloudflare.com/d1/)
- **Deployment:** [Cloudflare Workers](https://pages.cloudflare.com/)
